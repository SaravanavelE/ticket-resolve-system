# SOP: Approved Software Installation

## Overview
Procedure for installing standard software.

## Steps
1. Verify software is in the Approved Software List (ASL).
2. If not, direct user to submit a Software Exception Request.
3. If approved, use SCCM or Intune Company Portal to push the installation.
4. If manual install is required, use CyberArk EPM or LAPS for temporary admin rights.
5. Do NOT give users permanent local admin rights.
