# Runbook: VPN Connectivity Issues

## Overview
Troubleshooting Cisco AnyConnect VPN drops.

## Steps
1. Ask user to check home internet connection.
2. Verify VPN service status on the internal dashboard.
3. Check if user is on the latest AnyConnect version (v4.10+).
4. Instruct user to clear VPN cache and restart client.
5. If issue persists, check firewall logs for the user's IP.
6. Escalate to Network Team if multiple users report drops.
